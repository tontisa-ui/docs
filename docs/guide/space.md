### space （自定义组件占位）
`不依赖于iview`

### 使用方式 (已注册全局组件)
```
 // 文件内直接使用
 <iv-space />
```
### props 属性
| 属性  |  说明  | 类型  | 默认值  |
| ------------ | ------------ | ------------ | ------------ |
| value  |  绑定的值，可使用 v-model 双向绑定  | Number, String  |  ''  |
| label | 左边label显示(为空时不渲染)	  | [String, Number]  |  ''  |
| required | 是否必填项（配合验证使用）	  | Boolean  |  false  |
|  placeholder | 占位文本 (已多语言化)		  | [String, Number]  |  -  |

### event 事件
| 属性  |  说明  | 返回值  |
| ------------ | ------------ | ------------ |
| on-focus | 输入框失去焦点时触发  |  value,event |


### slot 插槽
| 属性  |  说明  |
| ------------ | ------------ |
| 无  |  占位  |
