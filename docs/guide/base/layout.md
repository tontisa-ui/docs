### layout （布局）
布局组件

### 使用方式 (已注册全局组件)
```
 // 文件内直接使用
 <iv-layout />
```
### props 属性
| 属性  |  说明  | 类型  | 默认值  |
| ------------ | ------------ | ------------ | ------------ |
| is-center | 是否垂直水平居中	  | Boolean  |  -  |
| head-height | 设置header高度，默认自适应	  | Number, String  |  'auto' |

### slot 插槽
| 属性  |  说明  |
| ------------ | ------------ |
| head  |  头部占位  |
| main  |  内容占位  |
| foot  |  底部占位  |
